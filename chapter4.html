Complete C Programming Guide

Comprehensive reference for C programming covering functions, storage classes, pointers, file handling, and advanced concepts for Class 12 Computer Science students.

1. Working with Functions

Functions are fundamental building blocks in C programming that enable code reusability, modularity, and organization. They allow you to break down complex problems into smaller, manageable pieces.

a. Function Definition

A function is a self-contained block of code designed to perform a specific task. It has a unique name, may accept parameters, and may return a value.

// Function definition syntax
return_type function_name(parameter_list) {
  // function body - code statements
  return value; // optional based on return type
}

b. Function Components

Every function in C consists of several key components that define its behavior and usage.

Function Prototype/Declaration

A function declaration informs the compiler about a function's name, return type, and parameters before it is defined. This allows the function to be called before its definition appears in the code.

// Function prototype examples
int add(int a, int b); // Declaration with parameters
void printMessage(); // Declaration without parameters
double calculateArea(double radius); // Declaration with different parameter type

Function Call

A function call invokes the function to execute its code. Arguments passed must match the parameters in type and order.

// Function call examples
int result = add(5, 3); // Call with arguments
printMessage(); // Call without arguments
double area = calculateArea(2.5); // Call with floating point argument

Function Definition

The function definition contains the actual implementation of the function - the code that will be executed when the function is called.

// Function definition examples
int add(int a, int b) {
  return a + b;
}

void printMessage() {
  printf("Hello, World!\n");
}

double calculateArea(double radius) {
  return 3.14159 * radius * radius;
}

Return Statement

The return statement terminates function execution and optionally returns a value to the calling function. The returned value must match the function's return type.

// Return statement examples
return 0; // Returning an integer
return result; // Returning a variable
return a + b; // Returning an expression
return; // For void functions (no value returned)

c. Types of Functions

Functions in C can be categorized based on their source and purpose.

Library Functions

Predefined functions provided by the C standard library. These are ready-to-use functions for common tasks.

// Examples of library functions
printf(); // Output formatting
scanf(); // Input reading
strlen(); // String length
sqrt(); // Square root
pow(); // Power calculation

User-Defined Functions

Functions created by programmers to perform specific tasks as needed for their applications.

// Examples of user-defined functions
int findMax(int a, int b);
void displayMenu();
float calculateTax(float income);
void sortArray(int arr[], int size);

Advantages of Using Functions

  • Code Reusability: Write once, use multiple times
  • Modularity: Break down complex problems into smaller parts
  • Abstraction: Hide implementation details
  • Maintainability: Easier to debug and update code
  • Collaboration: Multiple programmers can work on different functions
  • Testing: Individual functions can be tested separately

Complete Function Example

// Function declaration (prototype)
int findMax(int num1, int num2);

int main() {
  int a = 10, b = 20, maximum;
  
  // Function call
  maximum = findMax(a, b);
  
  printf("Maximum between %d and %d is %d\n", a, b, maximum);
  
  return 0;
}

// Function definition
int findMax(int num1, int num2) {
  if (num1 > num2) {
    return num1;
  } else {
    return num2;
  }
}

2. Categories of Functions with Examples

Functions in C can be categorized based on whether they return a value and whether they accept parameters. Understanding these categories helps in designing efficient and modular programs.

i. With return type but no arguments

These functions return a value but do not accept any parameters. They are useful for getting data or status without external input.

// Example: Function that returns a value without parameters
int getRandomNumber() {
  return rand() % 100; // Returns random number between 0-99
}

// Example: Function to get user input
int getUserAge() {
  int age;
  printf("Enter your age: ");
  scanf("%d", &age);
  return age;
}

// Call: int random = getRandomNumber();
// Call: int age = getUserAge();

ii. With return type with arguments

These functions both return a value and accept parameters. They are the most common type, performing calculations or operations on input data.

// Example: Function with return type and parameters
float calculateCircleArea(float radius) {
  return 3.14159 * radius * radius;
}

// Example: Function to find maximum of three numbers
int findMaxOfThree(int a, int b, int c) {
  if (a >= b && a >= c) return a;
  else if (b >= a && b >= c) return b;
  else return c;
}

// Call: float area = calculateCircleArea(5.5);
// Call: int max = findMaxOfThree(10, 25, 15);

iii. No return type with no arguments

These functions neither return a value nor accept parameters. They are typically used for displaying information or performing actions without needing input/output.

// Example: Function with no return type and no parameters
void displayWelcomeMessage() {
  printf("************************************\n");
  printf("* *\n");
  printf("* Welcome to Our Program! *\n");
  printf("* *\n");
  printf("************************************\n");
}

// Example: Function to clear screen
void clearScreen() {
  // System-specific command to clear screen
  system("cls"); // For Windows
  // system("clear"); // For Linux/Mac
}

// Call: displayWelcomeMessage();
// Call: clearScreen();

iv. No return type with arguments

These functions don't return a value but accept parameters. They are used for performing actions that require input but don't need to return data.

// Example: Function with no return type but with parameters
void printMultiplicationTable(int num) {
  printf("Multiplication table for %d:\n", num);
  for (int i = 1; i <= 10; i++) {
    printf("%d x %d = %d\n", num, i, num * i);
  }
}

// Example: Function to display student details
void displayStudentInfo(char name[], int age, float gpa) {
  printf("Student Name: %s\n", name);
  printf("Age: %d\n", age);
  printf("GPA: %.2f\n", gpa);
}

// Call: printMultiplicationTable(7);
// Call: displayStudentInfo("John Doe", 20, 3.75);

Complete Program Example

// Program demonstrating all function categories
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

// Function prototypes
int getRandomNumber(); // Category 1
float calculateAverage(int a, int b, int c); // Category 2
void displayHeader(); // Category 3
void printSquare(int num); // Category 4

int main() {
  srand(time(0)); // Seed random number generator
  
  displayHeader(); // Category 3 function call
  
  int random = getRandomNumber(); // Category 1 function call
  printf("Random number: %d\n", random);
  
  printSquare(random); // Category 4 function call
  
  float avg = calculateAverage(10, 20, 30); // Category 2 function call
  printf("Average of 10, 20, 30: %.2f\n", avg);
  
  return 0;
}

// Category 1: With return type but no arguments
int getRandomNumber() {
  return rand() % 100 + 1; // Returns random number between 1-100
}

// Category 2: With return type with arguments
float calculateAverage(int a, int b, int c) {
  return (a + b + c) / 3.0;
}

// Category 3: No return type with no arguments
void displayHeader() {
  printf("====================================\n");
  printf(" FUNCTION DEMONSTRATION \n");
  printf("====================================\n");
}

// Category 4: No return type with arguments
void printSquare(int num) {
  printf("Square of %d is %d\n", num, num * num);
}

3. Storage Classes

Storage classes in C define the scope, visibility, and lifetime of variables and functions. They determine where the variable is stored and how long it exists.

Storage Class Keyword Lifetime Scope Initial Value
Automatic auto Function block Local to function Garbage
External extern Entire program Global Zero
Register register Function block Local to function Garbage
Static static Entire program Local to function Zero

i. Automatic (auto)

Default storage class for local variables. Variables are created when the function is called and destroyed when the function exits.

void func() {
  auto int x = 10; // same as: int x = 10;
}

ii. External (extern)

Used to declare a global variable that is visible to all program files. Defined in one file and can be used in others.

// File1.c
int count = 0; // Definition

// File2.c
extern int count; // Declaration
printf("%d", count);

iii. Register

Suggests to the compiler to store the variable in a CPU register instead of RAM for faster access.

void func() {
  register int i; // For loop counters
}

iv. Static

Preserves variable value between function calls. For global variables, limits scope to the current file.

void counter() {
  static int count = 0; // Preserves value
  count++;
  printf("%d ", count);
}
// First call: 1, Second: 2, Third: 3

4. Recursive Functions

A recursive function is a function that calls itself directly or indirectly. Recursion is useful for solving problems that can be broken down into smaller, similar subproblems.

Syntax of Recursive Function

return_type function_name(parameters) {
    // Base case
    if (base_condition) 
        return value;
    
    // Recursive case
    return function_name(modified_parameters);
}

Example: Factorial Calculation

int factorial(int n) {
  // Base case
  if (n == 0 || n == 1)
    return 1;
  
  // Recursive case
  return n * factorial(n - 1);
}

// Call: int result = factorial(5); // 120

Example: Fibonacci Series

int fibonacci(int n) {
  // Base cases
  if (n <= 1)
    return n;
  
  // Recursive case
  return fibonacci(n - 1) + fibonacci(n - 2);
}

// Call: int fib = fibonacci(6); // 8

Example: Tower of Hanoi

void towerOfHanoi(int n, char from_rod, char to_rod, char aux_rod) {
  if (n == 1) {
    printf("Move disk 1 from rod %c to rod %c\n", from_rod, to_rod);
    return;
  }
  towerOfHanoi(n - 1, from_rod, aux_rod, to_rod);
  printf("Move disk %d from rod %c to rod %c\n", n, from_rod, to_rod);
  towerOfHanoi(n - 1, aux_rod, to_rod, from_rod);
}

// Call: towerOfHanoi(3, 'A', 'C', 'B');

Advantages and Disadvantages

Advantages Disadvantages
Elegant solution for problems with recursive nature May use more memory due to function call stack
Reduces code length for certain problems Can be slower than iterative solutions
Easier to write and understand for some problems Risk of stack overflow for deep recursion
Direct implementation of mathematical definitions Harder to debug for beginners

5. Passing Arrays to Functions

Arrays can be passed to functions either by passing individual elements or by passing the entire array. When passing an entire array, we typically pass the base address and size.

Example 1: Passing 1D Array

// Function to print array elements
void printArray(int arr[], int size) {
  for (int i = 0; i < size; i++) {
    printf("%d ", arr[i]);
  }
  printf("\n");
}

int main() {
  int numbers[] = {1, 2, 3, 4, 5};
  printArray(numbers, 5);
  return 0;
}

Example 2: Passing 2D Array

// Function to print 2D array
void printMatrix(int rows, int cols, int mat[][cols]) {
  for (int i = 0; i < rows; i++) {
    for (int j = 0; j < cols; j++) {
      printf("%d ", mat[i][j]);
    }
    printf("\n");
  }
}

int main() {
  int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};
  printMatrix(2, 3, matrix);
  return 0;
}

Example 3: Modifying Array Elements

// Function to double array elements
void doubleArray(int *arr, int size) {
  for (int i = 0; i < size; i++) {
    arr[i] *= 2; // Modifies original array
  }
}

int main() {
  int nums[] = {1, 2, 3};
  doubleArray(nums, 3);
  // nums now contains {2, 4, 6}
  return 0;
}

Example 4: Array Operations

// Function to find sum of array elements
int arraySum(int arr[], int size) {
  int sum = 0;
  for (int i = 0; i < size; i++) {
    sum += arr[i];
  }
  return sum;
}

// Function to find maximum element
int findMax(int arr[], int size) {
  int max = arr[0];
  for (int i = 1; i < size; i++) {
    if (arr[i] > max) {
      max = arr[i];
    }
  }
  return max;
}

int main() {
  int values[] = {5, 2, 9, 1, 7};
  int total = arraySum(values, 5);
  int maximum = findMax(values, 5);
  printf("Sum: %d, Max: %d\n", total, maximum);
  return 0;
}

6. Structures and Unions

Structures and unions are user-defined data types that allow grouping of variables of different types under a single name.

a. Structures

A structure is a collection of variables of different types under a single name. Each element in a structure is called a member.

// Defining a structure
struct Student {
  int id;
  char name[50];
  float marks;
};

// Declaring structure variable
struct Student s1;

// Accessing structure members
s1.id = 101;
strcpy(s1.name, "John Doe");
s1.marks = 85.5;

b. Unions

A union is similar to a structure but all members share the same memory location. Only one member can contain a value at any given time.

// Defining a union
union Data {
  int i;
  float f;
  char str[20];
};

// Declaring union variable
union Data data;

// Only one member can hold a value at a time
data.i = 10;
// data.f and data.str will not contain valid values now
Feature Structure Union
Memory Allocation Each member has its own memory All members share the same memory
Size Sum of sizes of all members Size of the largest member
Access All members can be accessed at any time Only one member can be accessed at a time
Use Case When you need to store different types of data together When you need to store only one of many data types

Example: Nested Structures

// Defining nested structures
struct Address {
  char city[50];
  char state[50];
  int pincode;
};

struct Employee {
  int id;
  char name[50];
  float salary;
  struct Address addr;
};

// Creating and accessing nested structure
struct Employee emp1;
emp1.id = 101;
strcpy(emp1.name, "John Doe");
emp1.salary = 50000.0;
strcpy(emp1.addr.city, "New York");
strcpy(emp1.addr.state, "NY");
emp1.addr.pincode = 10001;

Example: Array of Structures

// Array of structures
struct Student {
  int id;
  char name[50];
  float marks;
};

int main() {
  struct Student class[5];
  
  // Input data for 5 students
  for (int i = 0; i < 5; i++) {
    printf("Enter details for student %d:\n", i+1);
    printf("ID: ");
    scanf("%d", &class[i].id);
    printf("Name: ");
    scanf("%s", class[i].name);
    printf("Marks: ");
    scanf("%f", &class[i].marks);
  }
  
  // Display student details
  printf("\nStudent Details:\n");
  for (int i = 0; i < 5; i++) {
    printf("ID: %d, Name: %s, Marks: %.2f\n",
           class[i].id, class[i].name, class[i].marks);
  }
  return 0;
}

7. Pointers

Pointers are variables that store memory addresses. They provide powerful capabilities for memory management and efficient data manipulation.

a. Pointer Basics

A pointer is a variable that stores the address of another variable. The address-of operator (&) returns the address of a variable, and the dereference operator (*) accesses the value at a pointer's address.

int var = 10;
int *ptr; // Pointer declaration
ptr = &var; // Assigning address to pointer

printf("Value of var: %d\n", var); // Output: 10
printf("Address of var: %p\n", &var); // Output: address
printf("Value of ptr: %p\n", ptr); // Output: address of var
printf("Value at ptr: %d\n", *ptr); // Output: 10

b. Pointers and Functions

Pointers can be used to pass variables by reference to functions, allowing functions to modify the original variables.

// Function to swap values using pointers
void swap(int *a, int *b) {
  int temp = *a;
  *a = *b;
  *b = temp;
}

int main() {
  int x = 5, y = 10;
  swap(&x, &y);
  printf("x = %d, y = %d", x, y); // Output: x = 10, y = 5
  return 0;
}

c. Pointers and Arrays

Pointers and arrays are closely related. The name of an array is a pointer to its first element.

int arr[5] = {1, 2, 3, 4, 5};
int *ptr = arr; // Points to first element

// Different ways to access array elements
printf("First element: %d\n", arr[0]);
printf("First element: %d\n", *ptr);
printf("Second element: %d\n", *(ptr + 1));
printf("Third element: %d\n", ptr[2]);

d. Pointers to Structures

Pointers can be used with structures to efficiently access and manipulate structure variables.

struct Student {
  int id;
  char name[50];
};

struct Student s1 = {101, "John"};
struct Student *ptr = &s1;

// Accessing structure members using pointer
printf("ID: %d\n", (*ptr).id); // Method 1
printf("Name: %s\n", ptr->name); // Method 2 (arrow operator)

Example: Dynamic Memory Allocation

#include <stdio.h>
#include <stdlib.h>

int main() {
  int n, i;
  int *arr;
  
  printf("Enter the number of elements: ");
  scanf("%d", &n);
  
  // Dynamically allocate memory
  arr = (int*)malloc(n * sizeof(int));
  
  if (arr == NULL) {
    printf("Memory allocation failed!\n");
    return 1;
  }
  
  // Input elements
  for (i = 0; i < n; i++) {
    printf("Enter element %d: ", i+1);
    scanf("%d", &arr[i]);
  }
  
  // Display elements
  printf("Array elements: ");
  for (i = 0; i < n; i++) {
    printf("%d ", arr[i]);
  }
  printf("\n");
  
  // Free allocated memory
  free(arr);
  
  return 0;
}

8. File Handling

File handling in C allows programs to store and retrieve data from files, enabling persistent data storage.

a. File Operations

The basic file operations include opening, reading, writing, and closing files. The FILE structure is used to represent a file stream.

// Basic file operations
FILE *filePointer;

// Opening a file
filePointer = fopen("filename.txt", "mode");

// Checking if file opened successfully
if (filePointer == NULL) {
  printf("File could not be opened\n");
  return 1;
}

// File operations (read/write)
// ...

// Closing the file
fclose(filePointer);

b. File Modes

Different file modes determine how a file is opened and what operations can be performed on it.

Mode Description
"r" Open for reading (file must exist)
"w" Open for writing (creates new file or truncates existing)
"a" Open for appending (creates new file if doesn't exist)
"r+" Open for reading and writing (file must exist)
"w+" Open for reading and writing (creates new file or truncates existing)
"a+" Open for reading and appending (creates new file if doesn't exist)

c. File Functions

C provides various functions for reading from and writing to files.

// Writing to a file
fprintf(filePointer, "Hello, World!\n");
fputs("This is a line\n", filePointer);

// Reading from a file
char buffer[100];
fgets(buffer, 100, filePointer); // Read a line
fscanf(filePointer, "%s", buffer); // Read a word

// Character I/O
fputc('A', filePointer); // Write a character
char ch = fgetc(filePointer); // Read a character

d. Random Access Files

C allows random access to files using functions like fseek(), ftell(), and rewind().

// Moving file pointer to specific position
fseek(filePointer, 10, SEEK_SET); // Move to 10th byte from beginning
fseek(filePointer, -5, SEEK_CUR); // Move 5 bytes back from current position
fseek(filePointer, -20, SEEK_END); // Move to 20th byte from end

// Getting current position
long position = ftell(filePointer);
printf("Current position: %ld\n", position);

// Moving to beginning of file
rewind(filePointer);

Example: File Copy Program

#include <stdio.h>

int main() {
  FILE *source, *target;
  char ch;
  
  source = fopen("source.txt", "r");
  if (source == NULL) {
    printf("Cannot open source file\n");
    return 1;
  }
  
  target = fopen("target.txt", "w");
  if (target == NULL) {
    fclose(source);
    printf("Cannot open target file\n");
    return 1;
  }
  
  // Copy file character by character
  while ((ch = fgetc(source)) != EOF) {
    fputc(ch, target);
  }
  
  printf("File copied successfully\n");
  
  fclose(source);
  fclose(target);
  
  return 0;
}

9. Typedef Keyword

The typedef keyword is used to create aliases for existing data types, making code more readable and manageable.

a. Basic Typedef Usage

typedef can be used to create shorter or more meaningful names for existing data types.

// Creating alias for existing data types
typedef unsigned long int ULINT;
typedef float REAL;

// Using the aliases
ULINT distance = 1000000;
REAL pi = 3.14159;

b. Typedef with Structures

typedef is commonly used with structures to avoid using the "struct" keyword repeatedly.

// Without typedef
struct Point {
  int x;
  int y;
};
struct Point p1; // Need to use 'struct' keyword

// With typedef
typedef struct {
  int x;
  int y;
} Point;
Point p2; // No need for 'struct' keyword

c. Typedef with Pointers

typedef can be used with pointers to create more readable pointer type definitions.

// Creating a pointer type
typedef int* IntPtr;

// Using the pointer type
int a = 10, b = 20;
IntPtr ptr1 = &a, ptr2 = &b;

printf("Values: %d, %d\n", *ptr1, *ptr2);

Advantages of Typedef

  • Makes code more readable and understandable
  • Simplifies complex type declarations
  • Provides abstraction from implementation details
  • Makes code easier to modify and maintain

Example: Using Typedef with Function Pointers

// Typedef for function pointer
typedef int (*Operation)(int, int);

// Some operations
int add(int a, int b) { return a + b; }
int subtract(int a, int b) { return a - b; }
int multiply(int a, int b) { return a * b; }

int main() {
  Operation op; // Function pointer variable
  int x = 10, y = 5, result;
  
  op = add;
  result = op(x, y);
  printf("Addition: %d\n", result);
  
  op = subtract;
  result = op(x, y);
  printf("Subtraction: %d\n", result);
  
  op = multiply;
  result = op(x, y);
  printf("Multiplication: %d\n", result);
  
  return 0;
}
